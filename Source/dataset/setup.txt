1 Dowload postgresql 9.6 preferably:
 1.1 for windows : https://www.enterprisedb.com/downloads/postgres-postgresql-downloads#windows
 1.2 for ubuntu  : https://www.enterprisedb.com/downloads/postgres-postgresql-downloads#linux
 1.3 for mac	 : https://www.enterprisedb.com/downloads/postgres-postgresql-downloads#mac

2 After installing postgresql , install postgis using stackbuilder.
 
  -open stackbuilder, then select postgresql 9.6 on port 5432
  -Downloading list , select spatial extension dropdown then tick PostGIS 2.3 Bundle for PostgreSQL 9.6 .

3 Now you can either open up pgAdmin and create database using pgAmin or cmd.

3.1 pgAdmin
 -under server there will be PostgreSQL 9.6
 -under PostgreSQL 9.6 there will be databases option. Right click to    
  create a new database. To add PostGIS as extension click on tools and select query tool then write following script:
  	CREATE EXTENSION postgis;
  check if postgis was succcessfully added using 
    SELECT postgis_full_version();
 - 

3.2 cmd for windows
	open up terminal and set environmental variables to C:\Program Files\PostgreSQL\9.6\bin.

	use command : 
				createdb -U postgres <DatabaseName>
				password: yourPostgreSQLpassword
				
				psql -U postgres -d <DatabaseName> -c "CREATE EXTENSION postgis"

4 To add shapefiles you can use either PostGIS Shapefile Import/Export Manager or cmd.

4.1 using PostGIS Shapefile Import/Export Manager 
 - Test connection using PostGIS Connection -- View Connection details
 - If connection successful , click add file. Navigate to directory of shapefiles. Insert SIRD of 26912 and adjust Geo Column.
 Click on Import then log Window will give result.

 PostGIS will create a table in PostGreSQL and you open pgAdmin and navigate to Database >> <DatabaseName> >> Schemas >> public >> Tables 
 to view the table of the shapefile added.

 4.2 using cmd in bin directory in C:\Program Files\PostgreSQL\9.6\bin.

 psql -U postgres -d <DatabaseName> -c "SELECT postgis_version()"
 
 use command:

 	shp2pgsql -I -s 26912 <Path/to/shapefile> public.(nameoftable) | psql -U postgres -d <DatabaseName> 

 	you open pgAdmin and navigate to Database >> <DatabaseName> >> Schemas >> public >> Tables to view the table of the shapefile added.
